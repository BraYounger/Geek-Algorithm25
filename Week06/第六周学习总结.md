# 第六周学习总结

## 1.动态规划

### 1.1.动态规划关键点

​	- 最优子结构opt[n] = best_of(opt[n-1], opt[n-2], ...)

​	- 存储中间状态: opt[i]

​	- 递推公式（状态转移方程/DP方程）

​	   例如，

​			Fib : opt[i] = opt[n-1] + opt[n-2];

​			二维路径 : opt[i, j] = opt\[i+1\]\[j\] + opt\[i\]\[j+1\]

### 1.2.小结

	 - 难点在于寻找最优子结构与定义DP方程
	 - 如果最优子结构是简单的一维数组且在DP方程中仅会使用固定的几个前置状态，则可以将DP方程简化为几个滚动变量用于存储中间状态，以优化空间复杂度至O(1)
